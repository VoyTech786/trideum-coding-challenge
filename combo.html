Combination Challenge

Explanation

This program combines the Fizz Buzz Challenge and the Fibonacci Sequence Challenge. The inputs are:

  • Divisor 1 and Divisor 2 from Fizz Buzz. Default values are Divisor 1=3 and Divisor 2=5.
  • Subsitution Phrase 1 and Phrase 2 from Fizz Buzz. Default values are Phrase 1="fizz" and Phrase 2="buzz".
  • Input X from Fibonacci Sequence.
  • Input Y and Input Z from Fibonacci Sequence. Default values are Input Y=1, and Input Z=2.

Note: Divisor 1, Divisor 2, Input X, Input Y, and Input Z are all limited to integers between 1 and 100, inclusive.

Using these inputs, this program will display the sequence up to F(x), except replaced with the Substitution Phrases you set.

Example: Setting Input X to 6 and using the default values yields: 1, 1, 2, fizz, buzz, 8.
