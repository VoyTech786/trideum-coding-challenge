Fibonacci Sequence Challenge

Explanation

The Fibonacci Sequence is the sequence of numbers such that F(n) = F(n-1) + F(n-2), where n is the nth number in the sequence. For example, F(6) = 8, and the sequence F(6) is 1,1,2,3,5,8. This program supports a more complex version of the Fibonacci Sequence: F(x) = F(x-y) + F(x-z), where Input X, Input Y, and Input Z are user inputs, limited to integers between 1 and 100, inclusive.
